# Microsoft Teams Connection

Connect Microsoft Teams to send messages and notifications to channels.

## Required Credentials

- **Webhook URL**

## Setup Steps

1. Open Microsoft Teams
2. Navigate to the channel where you want to send messages
3. Click the three dots (...) next to the channel name
4. Select "Connectors" (or "Workflows" in newer versions)
5. Find "Incoming Webhook" and click "Configure"
6. Name your webhook and optionally add an image
7. Click "Create"
8. Copy the Webhook URL
9. Add it to your Workflow Builder connections

## Features

- Send messages to Teams channels
- Send formatted cards with buttons
- Include images and links
- Trigger notifications from workflows

## Best Practices

- Use descriptive webhook names for easy management
- Test webhooks in a private channel first
- Monitor webhook usage for security
- Rotate webhooks periodically
