# Linear Connection

Connect Linear to create issues, manage projects, and automate engineering workflows.

## Required Credentials

- **API Key**

## Setup Steps

1. Go to [Linear Settings](https://linear.app/settings/api)
2. Click "Create key"
3. Name your API key
4. Select the scope (Personal or Workspace)
5. Click "Create"
6. Copy the generated API key
7. Add it to your Workflow Builder connections

## Features

- Create issues in teams
- Update issue status and priority
- Assign issues to team members
- Add comments to issues
- Query issues with filters

## Best Practices

- Use team IDs for consistent issue creation
- Handle issue state transitions properly
- Monitor GraphQL query complexity
- Use labels for better organization
