# Trello Connection

Connect Trello to create cards, manage boards, and automate task workflows.

## Required Credentials

- **API Key**
- **Token**

## Setup Steps

1. Go to [Trello Power-Ups Admin](https://trello.com/power-ups/admin)
2. Click "New" to create a new Power-Up (or use existing)
3. Fill in the required details
4. Navigate to the API Key section
5. Copy your API Key
6. Click the "Token" link to generate a token
7. Authorize the application
8. Copy the generated token
9. Add both to your Workflow Builder connections

## Features

- Create cards on boards
- Move cards between lists
- Update card details
- Add comments and attachments
- List boards, lists, and cards

## Best Practices

- Use board and list IDs for consistent operations
- Handle card due dates with proper timezone
- Monitor API rate limits (100 requests per 10 seconds)
- Use webhooks for real-time updates when needed
