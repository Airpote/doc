# Grok Connection

Connect to xAI's Grok AI for advanced reasoning and analysis capabilities.

## Required Credentials

- **API Key** (from xAI platform)

## Setup Steps

1. Visit [x.ai](https://x.ai/)
2. Sign up for an account
3. Navigate to the API section
4. Generate a new API key
5. Copy the API key
6. Add the API key to your Workflow Builder connections

## Features

- Advanced reasoning
- Real-time information access
- Humorous and helpful responses
- Integration with X (Twitter) data

## Use Cases

- Complex problem solving
- Data analysis
- Creative content generation
- Automated research
- Intelligent automation with personality

## Notes

Grok is built by xAI and may have different capabilities compared to other AI models. Check xAI's documentation for the latest features.
