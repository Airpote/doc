# Replicate Connection

Connect Replicate to run AI models from the community with simple API calls.

## Required Credentials

- **API Key**

## Setup Steps

1. Go to [Replicate](https://replicate.com/)
2. Create an account or sign in
3. Navigate to Account Settings â†’ API tokens
4. Copy your API token
5. Add it to your Workflow Builder connections

## Features

- Run image generation models (Stable Diffusion, SDXL, etc.)
- Run language models
- Run audio models
- Run video models
- Access thousands of community models

## Supported Use Cases

- Image generation
- Image editing
- Text-to-speech
- Speech-to-text
- Video generation
- Custom model deployments

## Best Practices

- Check model documentation for required inputs
- Monitor prediction costs
- Use webhooks for long-running predictions
- Handle cold start times for less popular models
