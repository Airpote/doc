# Alchemy Connection

Connect Alchemy for reliable blockchain infrastructure and APIs.

## Required Credentials

- **API Key**

## Setup Steps

1. Go to [Alchemy Dashboard](https://dashboard.alchemy.com/)
2. Create an account or sign in
3. Create a new app or use existing
4. Select your target blockchain(s)
5. Copy your API key from the app details
6. Add the API key to your Workflow Builder connections

## Supported Chains

- Ethereum (mainnet & testnets)
- Polygon
- Arbitrum
- Optimism
- And many more

## Features

- Enhanced RPC endpoints
- NFT API access
- Token API access
- Webhook notifications
- Historical data queries

## Best Practices

- Create separate apps for different environments
- Monitor compute units usage
- Use websockets for real-time data
- Handle rate limits with exponential backoff
