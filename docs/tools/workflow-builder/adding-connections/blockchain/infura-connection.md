# Infura Connection

Connect Infura for reliable Ethereum and IPFS infrastructure.

## Required Credentials

- **Project ID**
- **Project Secret** (optional, for added security)

## Setup Steps

1. Go to [Infura Dashboard](https://infura.io/)
2. Create an account or sign in
3. Create a new project
4. Select "Web3 API" as the product
5. Copy your Project ID
6. Optionally enable and copy the Project Secret
7. Add credentials to your Workflow Builder connections

## Supported Chains

- Ethereum (mainnet & testnets)
- Polygon
- Arbitrum
- Optimism
- Avalanche
- And more

## Features

- JSON-RPC endpoints
- WebSocket connections
- IPFS gateway access
- Transaction relay
- Archive data access

## Best Practices

- Enable project secret for production
- Monitor daily request counts
- Use appropriate network endpoints
- Handle rate limits gracefully
- Consider upgrading for higher limits
