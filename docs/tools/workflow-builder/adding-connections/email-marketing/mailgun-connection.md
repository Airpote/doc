# Mailgun Connection

Connect Mailgun to send and receive emails with powerful API features.

## Required Credentials

- **API Key**
- **Domain** (your sending domain)

## Setup Steps

1. Go to [Mailgun Dashboard](https://app.mailgun.com/)
2. Create an account or sign in
3. Add and verify a domain (or use sandbox for testing)
4. Go to API Keys section
5. Copy your Private API key
6. Note your verified domain
7. Add credentials to your Workflow Builder connections

## Features

- Send emails via API
- Receive emails via webhooks
- Track email events
- Manage mailing lists
- Handle bounces and complaints

## Best Practices

- Use a verified custom domain for production
- Set up webhooks for delivery tracking
- Monitor reputation metrics
- Handle bounces and unsubscribes automatically
