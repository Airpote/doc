# Mailchimp Connection

Connect Mailchimp to manage subscribers and send marketing campaigns.

## Required Credentials

- **API Key**
- **Server** (data center prefix)

## Setup Steps

1. Go to [Mailchimp](https://mailchimp.com/)
2. Sign in to your account
3. Click your profile icon → Account → Extras → API keys
4. Click "Create A Key"
5. Copy the API key
6. Note the server prefix (e.g., `us21` from your API key ending `-us21`)
7. Add credentials to your Workflow Builder connections

## Features

- Add subscribers to lists
- Send campaigns
- Manage audience segments
- Create automations
- Track campaign performance

## Best Practices

- Use double opt-in for subscribers
- Segment your audiences for targeted campaigns
- Monitor engagement metrics
- Comply with email marketing regulations
