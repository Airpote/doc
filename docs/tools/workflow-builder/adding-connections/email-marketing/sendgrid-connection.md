# SendGrid Connection

Connect SendGrid to send transactional and marketing emails.

## Required Credentials

- **API Key**
- **From Email** (optional default sender)

## Setup Steps

1. Go to [SendGrid Dashboard](https://app.sendgrid.com/)
2. Create an account or sign in
3. Go to Settings → API Keys
4. Click "Create API Key"
5. Name your key and select permissions:
   - Full Access, or
   - Restricted Access with "Mail Send" enabled
6. Click "Create & View"
7. Copy the API key (shown only once)
8. Verify a sender identity in Settings → Sender Authentication
9. Add credentials to your Workflow Builder connections

## Features

- Send transactional emails
- Send emails with templates
- Add attachments
- Track opens and clicks
- Manage contact lists

## Best Practices

- Verify sender domains for better deliverability
- Use templates for consistent branding
- Monitor bounce rates and spam reports
- Handle unsubscribes properly
