# Resend Connection

Connect Resend for modern email delivery with excellent developer experience.

## Required Credentials

- **API Key**

## Setup Steps

1. Go to [Resend Dashboard](https://resend.com/)
2. Create an account or sign in
3. Go to API Keys section
4. Click "Create API Key"
5. Name your key and set permissions
6. Copy the API key
7. Add a domain in Domains section (optional but recommended)
8. Add credentials to your Workflow Builder connections

## Features

- Send transactional emails
- React Email integration
- Domain verification
- Email analytics
- Webhook notifications

## Best Practices

- Verify your domain for better deliverability
- Use the free tier for development
- Monitor delivery rates
- Use proper from addresses
