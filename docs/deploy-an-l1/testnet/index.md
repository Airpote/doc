# Deploy a Testnet L1

## Overview

Deploy a testnet L1 for development and testing with AmiChain's managed service.

| Nodes | 1 validator node |

## Included Features

Testnet packages include:

- Hosting & maintenance of a block explorer customized for your L1
- Support & maintenance by our team
- Monitoring
- Custom alerting
- Dedicated public RPC endpoint
- Configurable Faucet for testing
